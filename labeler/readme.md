# $mol_labeler

The component that provide labels before the content.

##[Online demo](http://eigenmethod.github.io/mol/#demo=mol_labeler_demo)

## Properties

`title() : string`
Property for setting title.

`content() : Array`
Property for placing the content.

## Usage examples
```
$mol_labeler
	title \Provider
	content / \ACME Provider Inc.
```
